# 공통 치환자

## 영역
- `<s_t3>`: 티스토리 공통 javascript가 삽입

## 기본설정
- `[##_title_##]`: 블로그 제목
- `page_title`: 페이지 설명
- `image`: 블로그 대표 이미지 url
- `blog_image`: 블로그 대표 이미지 tag
- `desc`: 블로그 설명
- `blogger`: 블로그 소유자의 필명

## URL
- `blog_link`: 블로그 url
- `rss_url`: rss feed 주소
- `taglog_link`: 태그로그 url
- `guestbook_link`: 방명록 url
- `owner_url`: 관리자 페이지 url
- `locallog_link`: 위치로그 url
